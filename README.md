*About the project:*
Managing bugs in a software project can quickly become chaotic without a proper system. This Bug Tracker provides a structured way to report issues, delegate responsibility, and monitor progress — all in one place.

*Features:*
User login and authentication
Log new bugs with a title, description, and priority level
Assign bugs to specific team members
Track the status of each bug (Open, In Progress, Resolved)
Dashboard overview of all active and resolved bugs

*Built with:* 
HTML / CSS / JavaScript /
Node.js
